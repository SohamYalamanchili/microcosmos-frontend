# Microcosmos Explorer

A simple, responsive web interface built to showcase microscopic discoveries for the Microcosmos community.

## Overview
This project is a front-end web application designed to present curated microscopic content in a clean and accessible format. It focuses on layout, responsiveness, and visual clarity rather than complex backend logic.

## Built With
- HTML
- TypeScript
- React
- Vite
- Tailwind CSS
